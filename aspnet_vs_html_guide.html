HTML Tags vs ASP.NET Web Forms Controls

This guide lists common HTML tags alongside their ASP.NET equivalents, helping you bridge client-side and server-side development.

Document Structure

HTML TagASP.NET EquivalentDescription
<html><html runat="server">Enables server-side rendering.
<head><head runat="server">Container for metadata, title, links, etc.
<body><body runat="server">Main body of the page.

Form and Input Controls

HTML TagASP.NET ControlDescription
<input type="text"><asp:TextBox />Text input field.
<input type="password"><asp:TextBox TextMode="Password" />Password input field.
<input type="checkbox"><asp:CheckBox />Checkbox input.
<input type="radio"><asp:RadioButton GroupName="..." />Radio button.
<input type="submit"><asp:Button />Form submit button.
<textarea><asp:TextBox TextMode="MultiLine" />Multiline text input.
<select><asp:DropDownList>Dropdown selector.

Lists and Tables

HTML TagASP.NET EquivalentDescription
<ul>, <ol>, <li>Use HTMLUnordered and ordered lists.
<table><asp:Table>Dynamic tables via code.
<tr>, <td><asp:TableRow>, <asp:TableCell>Table structure in ASP.NET.
<th><asp:TableHeaderCell>Header cell.

Navigation and Media

HTML TagASP.NET EquivalentDescription
<a href="..."><asp:HyperLink NavigateUrl="..." />Navigation link.
<img src="..."><asp:Image ImageUrl="..." />Image element.
<video>, <audio>Use HTMLMultimedia support.

Layout and Containers

HTML TagASP.NET EquivalentDescription
<div><asp:Panel /> or <div runat="server">Content container.
<span><asp:Label />Inline container.

ASP.NET Specific Controls

ControlDescription
<asp:GridView>Display tabular data with sorting and paging.
<asp:Repeater>Custom template-based data display.
<asp:DataList>Repeating list control.
<asp:Calendar>Calendar UI control.
<asp:FileUpload>File upload control.
<asp:Login>Authentication login form.

Example: ASP.NET Registration Form

<form id="form1" runat="server"> <asp:Label ID="lblName" Text="Name:" runat="server" /><br /> <asp:TextBox ID="txtName" runat="server" /><br /> <asp:Label ID="lblEmail" Text="Email:" runat="server" /><br /> <asp:TextBox ID="txtEmail" runat="server" /><br /> <asp:Button ID="btnSubmit" Text="Submit" runat="server" /> </form>

Tip: You can copy and paste this code into a `.aspx` file in Visual Studio and run it to see how it works.
